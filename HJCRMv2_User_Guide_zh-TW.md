# Hour Jungle CRM v2 使用說明

版本：v2
更新日期：2026-01-02

這份文件會用最簡單的步驟，教你完成每天的工作。

---

## 1. 先登入
1. 打開：`https://hj-v2.yourspce.org`
2. 輸入帳號密碼
3. 看到首頁（Dashboard）就代表成功

---

## 2. 首頁（Dashboard）
登入後第一個看到的頁面，顯示：
- 本月待處理事項數量
- 近期到期合約提醒
- 逾期繳費統計

這裡可以快速掌握今天要做什麼。

---

## 3. 你會看到的主要功能

| 功能 | 用途 |
|------|------|
| 合約 | 管理合約資料 |
| 續約 | 看快到期的合約、處理續約 |
| 繳費 | 記錄付款、催繳 |
| 發票 | 開立或作廢發票 |
| 客戶 | 管理客戶資料 |

---

## 4. 新增合約（最常用）
1. 進入「合約」
2. 點「新增合約」
3. 填寫客戶資料與合約內容
4. 按「儲存」
5. 成功後會自動產生繳費記錄

小提醒：
- 如果是續約，系統會把舊合約資料帶過來，你只要改新的條款

---

## 5. 續約合約
1. 進入「續約」或「合約」
2. 找到要續約的合約
3. 點「建立續約合約」
4. 系統會帶入舊合約內容
5. 確認新條款後儲存

### 續約頁面小技巧
- **急件**：顯示 7 天內到期的合約（優先處理）
- **進行中**：顯示正在處理的續約
- **已完成**：顯示處理完畢的項目
- **已移交**切換：顯示/隱藏已完成交接的項目

---

## 6. 記錄繳費
1. 進入「繳費」
2. 找到客戶（可用搜尋框）
3. 點「記錄繳費」
4. 填金額、日期、付款方式
5. 按「儲存」

---

## 7. 開立或作廢發票
### 開立
1. 進入「發票」
2. 找到待開立項目
3. 點「開立發票」
4. 填入發票資訊

### 作廢
1. 找到已開立的發票
2. 點「作廢」
3. 輸入原因即可

---

## 8. 合約到期／終止
- 合約到期：在「續約」頁會提醒
- 合約終止：在合約頁點「終止合約」即可

---

## 9. 常見問題

| 問題 | 解決方式 |
|------|----------|
| 看不到資料 | 重新整理頁面（F5 或 Cmd+R） |
| 欄位不確定怎麼填 | 先問主管或管理者 |
| 不確定要做什麼 | 先看「續約」或「繳費」頁的提醒 |
| 找不到客戶 | 用搜尋框輸入公司名或客戶名 |

---

## 10. 用一句話記住
有問題就回到「合約 → 續約 → 繳費 → 發票」這條主線做就對了。
