## 六、查詢指令參考

### 查詢特定公司的合約
```sql
SELECT c.id as contract_id, cu.id as customer_id, cu.company_name, c.status
FROM contracts c
JOIN customers cu ON c.customer_id = cu.id
WHERE cu.company_name LIKE '%關鍵字%'
ORDER BY c.status, c.id;
```

### 查詢所有有效合約（尚未設定位置）
```sql
SELECT c.id, cu.company_name, c.position_number
FROM contracts c
JOIN customers cu ON c.customer_id = cu.id
WHERE c.status = 'active' AND c.branch_id = 1 AND c.position_number IS NULL
ORDER BY cu.company_name;
```

### 檢視目前位置分配狀態
```sql
SELECT c.position_number, cu.company_name, c.id as contract_id
FROM contracts c
JOIN customers cu ON c.customer_id = cu.id
WHERE c.status = 'active' AND c.branch_id = 1 AND c.position_number IS NOT NULL
ORDER BY c.position_number;
```
