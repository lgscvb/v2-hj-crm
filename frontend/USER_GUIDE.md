## 系統是什麼？

### 簡單來說...

想像你有一本超級聰明的筆記本，它可以：
- 📝 記住所有客戶的資料
- 💰 提醒你誰該繳錢了
- 📊 自動算出這個月賺了多少錢
- 🔔 告訴你哪些合約快到期了

這就是 **Hour Jungle CRM**（客戶關係管理系統）！

## 功能介紹

### 1. 👥 客戶管理

這裡可以看到所有客戶的資料。

#### 怎麼找客戶？

```
┌─────────────────────────────────────────────┐
│  🔍 搜尋框                                   │
│  ┌─────────────────────────────────────┐    │
│  │ 輸入客戶姓名或公司名稱...            │    │
│  └─────────────────────────────────────┘    │
└─────────────────────────────────────────────┘
```

1. 在搜尋框輸入客戶名字（例如：王小明）
2. 系統會自動找出符合的客戶

#### 怎麼篩選客戶？

```
分館篩選：[全部 ▼]  [大忠館]  [環瑞館]
狀態篩選：[全部 ▼]  [活躍]  [休眠]  [流失]
```

點一下按鈕就可以篩選！

#### 欄位選擇器（新功能！）

不想看太多資料？點擊「📋 欄位」按鈕，勾選你想看的欄位：

```
☑️ 姓名
☑️ 公司名稱
☑️ 電話
☐ Email（不勾就不會顯示）
☑️ 分館
```

---

### 2. 📄 合約管理

管理所有客戶的合約。

#### 合約狀態說明：

| 狀態 | 意思 | 顏色 |
|------|------|------|
| 🟢 生效中 | 合約正在執行 | 綠色 |
| 🟡 即將到期 | 30天內會到期 | 黃色 |
| 🔴 已到期 | 合約已經過期 | 紅色 |
| ⚫ 已終止 | 合約已結束 | 灰色 |

#### 怎麼看合約詳情？

點一下合約編號（例如：DZ-001-2025），就會看到詳細資料！

---

### 3. 💳 繳費管理

記錄客戶的繳費情況。

#### 繳費狀態說明：

```
🟢 已繳費 (paid)     → 錢已經收到了！
🟡 待繳費 (pending)  → 還沒到繳費日，等待中
🔴 逾期 (overdue)    → 過了繳費日還沒繳！
```

#### 怎麼記錄收款？

1. 找到要記錄的那筆繳費
2. 點「收款」按鈕
3. 選擇收款方式（現金/轉帳/信用卡）
4. 完成！

```
┌─────────────────────────────────┐
│  收款確認                        │
├─────────────────────────────────┤
│  客戶：王小明                    │
│  金額：$3,600                    │
│  收款方式：[轉帳 ▼]              │
│                                  │
│  [取消]        [確認收款]        │
└─────────────────────────────────┘
```

### 5. 📊 報表中心

這裡可以看各種統計報表。

#### 報表類型：

| 報表 | 用途 |
|------|------|
| 📈 營收報表 | 看收了多少錢 |
| 📊 趨勢分析 | 看營收是上升還是下降 |
| 💵 財務分析 | 看收款率、逾期率 |
| 👥 客戶分析 | 看客戶組成 |
| � 合約分析 | 看合約分布 |
| ⚠️ 逾期報表 | 看誰還沒繳錢 |
| 💼 佣金報表 | 看要付多少佣金 |

#### 時間篩選（新功能！）

```
[月] [季] [年]     [2025-12 ▼]     YoY +15.2%
```

- **月**：看單月數據
- **季**：看整季數據
- **年**：看整年數據
- **YoY**：跟去年同期比較（Year over Year）
- **MoM**：跟上個月比較（Month over Month）
- **QoQ**：跟上一季比較（Quarter over Quarter）

### 6. 💼 佣金管理

管理介紹人的佣金。

#### 佣金計算方式：

```
佣金 = 月租金 × 1個月
例如：客戶月租 $3,600 → 佣金 = $3,600
```

#### 佣金狀態：

| 狀態 | 意思 |
|------|------|
| 待審核 | 還在等待確認 |
| 可付款 | 可以付佣金了 |
| 已付款 | 佣金已經付了 |

## 常見問題

### Q1：找不到某個客戶怎麼辦？

**A**：試試以下方法：
1. 確認搜尋的名字有沒有打錯
2. 用公司名稱搜尋看看
3. 把篩選條件改成「全部」

### Q2：數字怎麼跟我算的不一樣？

**A**：系統的數字是即時從資料庫計算的。如果有疑問：
1. 檢查日期範圍是否正確
2. 確認篩選條件
3. 刷新頁面試試（按 F5 或 Cmd+R）

### Q3：合約狀態怎麼改？

**A**：
1. 進入「合約管理」
2. 找到要修改的合約
3. 點擊合約進入詳情頁
4. 修改狀態後儲存

### Q4：怎麼記錄客戶已經繳費？

**A**：
1. 進入「繳費管理」
2. 找到對應的繳費記錄
3. 點「收款」按鈕
4. 選擇收款方式
5. 確認

### Q5：頁面跑很慢怎麼辦？

**A**：
1. 刷新頁面（F5 或 Cmd+R）
2. 清除瀏覽器快取
3. 試試用 Chrome 瀏覽器
