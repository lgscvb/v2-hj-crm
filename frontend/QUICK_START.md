# 🚀 Hour Jungle CRM 快速入門

> 5 分鐘學會使用系統！

---

## 第 1 分鐘：打開系統

```
瀏覽器輸入 → https://hj.yourspce.org → 按 Enter
```

✅ 完成！你已經進入系統了

---

## 第 2 分鐘：認識首頁

首頁有 4 個重要數字：

```
┌─────────────────────────────────────────────┐
│                                             │
│   💰 本月營收      👥 活躍客戶              │
│   這個月收多少     有幾個客戶               │
│                                             │
│   📄 有效合約      ⏰ 待收款                │
│   幾份合約在跑     還要收多少               │
│                                             │
└─────────────────────────────────────────────┘
```

---

## 第 3 分鐘：學會找客戶

**方法 1：搜尋**
```
🔍 輸入名字 → 找到客戶！
```

**方法 2：篩選**
```
點 [大忠館] → 只看大忠館的客戶
點 [環瑞館] → 只看環瑞館的客戶
```

---

## 第 4 分鐘：學會看合約

合約狀態一看就懂：

| 看到這個 | 代表 |
|----------|------|
| 🟢 生效中 | 合約正常 |
| 🟡 即將到期 | 快到期了，要提醒客戶 |
| 🔴 已到期 | 已經過期 |
| ⚫ 已終止 | 合約結束了 |

---

## 第 5 分鐘：學會收款

收到客戶的錢後：

```
1. 點「💳 繳費管理」
2. 找到客戶
3. 點「收款」按鈕
4. 選收款方式（現金/轉帳）
5. 按確認

✅ 完成！
```

---

## 🎯 每日必看功能

### 早上上班先看：

1. **儀表板** → 看整體狀況
2. **繳費管理** → 看今天誰要繳費
3. **續約提醒** → 看誰的合約快到期

### 月底要看：

1. **報表中心** → 看這個月營收
2. **逾期報表** → 看誰還沒繳錢

---

## ❓ 遇到問題？

### 頁面怪怪的？
→ 按 `Cmd + R`（Mac）或 `F5`（Windows）重新整理

### 找不到客戶？
→ 把篩選改成「全部」再搜尋

### 數字不對？
→ 確認日期範圍，再重新整理頁面

### 還是不會？
→ 點「🤖 AI 助手」直接問！

## 🌟 小技巧

1. **想看更多資料？** → 往下滾動頁面
2. **想看特定欄位？** → 點「📋 欄位」選擇
3. **想比較不同月？** → 在報表中心切換月份
4. **想知道漲跌？** → 看 YoY/MoM 數字
